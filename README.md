# Proteomics-analysis-tutorial
Beginner-friendly tutorial on FragPipe and MetaboAnalyst

This is a tutorial for processing RAW proteomics data from LC-MS/MS using FragPipe, followed by statistical analysis using MetaboAnalyst. 

## Learning Objectives
- Understand the basic principles of LC-MS/MS Proteomics
- Install and configure FragPipe
- Run peptide and protein identification
- Perform label-free protein quantification (LFQ)
- Understand MaxLFQ and Match Between Runs (MBR)
- Prepare data for MetaboAnalyst
- Perform basic statistical analysis on MetaboAnalyst

## Outline
Tutorial contents for FragPipe:
- Introduction to proteomics
- LC-MS/MS workflow
- Installing and configuring FragPipe
- Running FragPipe
- Understanding FragPipe parameters
- Protein identification and quantification
- Quality control
- Troubleshooting

Tutorial contents for MetaboAnalyst:
- Preparing input files from FragPipe output
- Data normalization and imputation
- PCA
- Heatmaps
- Volcano plots
- Differential protein analysis
- Functional enrichment

## Example Dataset
For this tutorial, we use a public dataset, Project PXD072757, title Comparative plasma proteomics of COVID-19 and Dengue patients with relevance to cardiovascular-associated pathways, available from https://www.ebi.ac.uk/pride/archive/projects/PXD072757. The dataset contains 14 samples.
